\#  Atividades de Lógica em C# - SP2



Este repositório contém os exercícios práticos desenvolvidos para o aprendizado de fundamentos de programação em C#, utilizando o \*\*Visual Studio Code\*\* e o \*\*.NET SDK\*\*.



\#  Estrutura das Atividades



&#x20;\*\*Atividade 1: Jogo de Adivinhação\*\* 🎲

&#x20;   \* Uso de `Random` para gerar números aleatórios e laços de repetição para tentativas do usuário.

&#x20;\*\*Atividade 2: Contador de Pares e Ímpares\*\* 🔢

&#x20;   \* Lógica matemática com o operador de módulo (`%`) para identificar e contar tipos de números.

&#x20;\*\*Atividade 3: Soma de Números Positivos\*\* ➕

&#x20;   \* Utilização de variável acumuladora e laço `while` com interrupção por número negativo.

&#x20;\*\*Atividade 4: Verificador de Números Primos\*\* 🔍

&#x20;   \* Algoritmo para contagem de divisores e verificação de primalidade.

&#x20;\*\*Atividade 5: Série de Fibonacci\*\* 🌀

&#x20;   \* Geração da sequência matemática baseada na soma dos dois termos anteriores.

&#x20;\*\*Atividade 6: Média de Temperaturas\*\* 🌡️

&#x20;   \* Uso de `Array` de tamanho fixo, cálculo de média e filtros com `foreach`.

&#x20;\*\*Atividade 7: Lista de Compras Inteligente\*\* 🛒

&#x20;   \* Manipulação de `List<string>`, ordenação alfabética com `.Sort()` e filtros de texto.



\#  Tecnologias e Conceitos Aplicados

&#x20;\*\*Linguagem:\*\* C# (.NET)

&#x20;\*\*Editor:\*\* Visual Studio Code

&#x20;\*\*Conceitos:\*\*

&#x20;   \* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`)

&#x20;   \* Conversão de tipos (`int.Parse`, `double.Parse` e `TryParse`)

&#x20;   \* Estruturas de Repetição (`for`, `while`, `foreach`)

&#x20;   \* Coleções de Dados (`Arrays` e `Lists`)
